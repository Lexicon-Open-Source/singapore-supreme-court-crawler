# Singapore Supreme Court Crawler


How to use:
```bash
 # Install Dependencies
 $ make install
 # Build
 $ go build -o singapore-supreme-court-crawler
 # Run Crawler and Scrapper
 $ ./singapore-supreme-court-crawler
 # Run Crawler
 $ ./singapore-supreme-court-crawler crawler
 # Run Scrapper
 $ ./singapore-supreme-court-crawler scrapper
```
